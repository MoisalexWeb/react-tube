## Run Locally using Git

Clone the project

```bash
  git clone https://github.com/MoisalexWeb/react-tube.git
```

Go to the project directory

```bash
  cd react-tube
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
